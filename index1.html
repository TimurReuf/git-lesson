  • git clone path to repository
  • git status - статус текущего локального репозитория
  • git add path-to-file - добавить новый файл под контроль ГИТА
  • git commit -am"added new file index1.html" - зафиксировать изменения и дать им комментарии
  • git push - отправить все закомичченные изменения в репозиторий на сервер
  • git pull - забрать свежия обновления
  • git branch name of branch-создаём ветку с названием
  • git checkout name of branch - переключаемся на ветку с нашим названием
